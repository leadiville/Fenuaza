# Fenuaza Premium Paint Store

Fenuaza is a modern e-commerce platform built for a premium paint brand, designed to make discovering, comparing, and purchasing paint products simple and intuitive.

The project focuses on creating a polished shopping experience while handling the practical needs of a commercial online store, including product management, custom product attributes, secure checkout, and payment integration.

## Overview

This project was developed using WordPress and WooCommerce with custom PHP, JavaScript, and CSS enhancements to extend the default functionality where required.

Instead of relying entirely on plugins, several features were customized to better match the client's business requirements and improve the overall user experience.

## Key Features

* Responsive e-commerce storefront
* Shop by colour experience
* Product variations for colours and sizes
* Custom product attribute management
* WooCommerce customization
* Paystack payment integration
* PayPal payment integration
* Category-based product navigation
* Optimized shopping workflow
* Mobile-friendly interface

## Technical Highlights
